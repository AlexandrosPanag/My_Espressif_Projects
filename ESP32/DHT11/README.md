DHT11
------------------------------


| DHT11 | ESP32 |
| --------------- | --------------- |
| Left Sensor Pin <--- | D4 (PIN 4) |
| VDD/VIN | DIGITAL 3V3 |
| Right Sensor Pin GND ---> | DIGITAL GND|
