DHT_ESP8266
-----

Temperature & humidity program. The program identifies the current temperature and then prints into the serial port of the esp8266 sensor


Connections:

| DHT11 | ESP32 |
| --------------- | --------------- |
| Left Sensor Pin <--- | D4 (GPIO 2) |
| VDD/VIN | 3V |
| Right Sensor Pin GND ---> | GND|
